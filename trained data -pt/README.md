# Trained Data Module

This directory contains the trained model for the Disease Predictor application.

## Contents

- **disease_model.pt**: This is the serialized PyTorch model file containing the weights and architecture information for the disease classification model.

## Usage

This model is intended to be loaded by the application's inference engine to predict diseases based on input symptoms or data. Ensure you have the compatible version of PyTorch installed to load this model successfully.
